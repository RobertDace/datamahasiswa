Tentang Sistem

Desain dan Tujuan

Halaman web ini dibuat sebagai implementasi tugas untuk menampilkan template layout dengan ukuran 1024x768 atau 800x600 pixel secara responsive, dilengkapi dengan komponen header, menu navigasi, konten utama, dan footer.

Desain menggunakan HTML, CSS, dan Bootstrap serta feather icon sebagai library pendukung. Menu header mendukung tampilan hamburger pada perangkat mobile seperti smartphone.
